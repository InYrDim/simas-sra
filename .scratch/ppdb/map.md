# Map: PPDB Form & Workflow

## Destination

A working UI prototype for the PPDB feature, including the Admin form builder (with system templates), the public student submission form, and the Admin review dashboard.

## Notes

- Domain: Education/School Admission (Penerimaan Peserta Didik Baru)
- Admin Form Builder UI: Simple field list, admin sets required/optional. Applying template clears existing fields then appends.
- Student UI: Multi-step, mobile-first.
- Dashboard UI: Data table with full filtering.
- Focus is on prototyping the UI.
