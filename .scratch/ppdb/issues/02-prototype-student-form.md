# 02 Prototype Student Public Form
Type: prototype
Status: resolved
Blocked by: 01

## Question
How should the public student form look and feel on mobile? It needs to be a multi-step wizard to be less intimidating. Let's build a prototype showing a typical 3-step process (e.g., Personal Info, Address, Documents) based on the output of the form builder.

## Answer
Variant A (Progress Bar Classic) was chosen. It uses a traditional multi-step wizard approach with a linear progress bar at the top, fading between steps, and clear Next/Back buttons at the bottom. This approach provides clear orientation for the user without overwhelming them with all fields at once.
